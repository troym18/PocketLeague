# Pocket Leauge 
## Description
Pocket League is a Rocket League inspired 2d game coded in Python using the CMU Graphics library. [Demo](https://youtu.be/9YCH4SF45Z4)
## Features
- Unique physics and collision
- 2 player version
- Jump mechanics
- Power-ups
- Training mode
- Robust UI/UX
## How to Run
Run main.py
